Battlegrounds was first released for Microsoft Windows via Steam's early access beta program in March 2017, with a full release in December 2017. 
The game was also released by Microsoft Studios for the Xbox One via its Xbox Game Preview program that same month, and officially released in September 2018. 
A free-to-play mobile version for Android and iOS was released in 2018, in addition to a port for the PlayStation 4. 
A version for the Stadia streaming platform was released in April 2020. Battlegrounds is one of the best-selling and most-played video games of all time. 
The game had sold over 70 million copies by 2020, with PUBG Mobile having been downloaded over 600 million times.
